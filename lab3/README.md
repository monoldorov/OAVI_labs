# Лабораторная работа №3. Фильтрация изображений и морфологические операции. Вариант 14
**Морфологическое расширение (дилатация). Структурирующий элемент — квадрат 3×3.**

## Теоретическая часть

В данной лабораторной работе рассматривается морфологическая операция **расширения (дилатации)** для бинарных изображений. По заданию для варианта 14 используется квадратный структурирующий элемент размером **3×3**. Матрица Omega имеет вид:


Matrix Omega:
1 1 1
1 1 1
1 1 1


Смысл дилатации состоит в том, что объект на бинарном изображении расширяется в соответствии с формой структурирующего элемента. В окрестности каждого пикселя рассматривается окно 3×3, и результат для центрального пикселя определяется по содержимому этого окна. Так как структурирующий элемент квадратный и полностью заполнен единицами, в обработке участвуют все 9 пикселей локальной окрестности.

В лабораторной работе в качестве входных данных используются бинарные изображения, а в качестве выходных данных необходимо получить:
1. обработанное изображение;
2. разностное изображение:
   - XOR для бинарных изображений;
   - модуль разности для полутоновых изображений.

Для наглядности полутоновая разность дополнительно усиливается умножением на 10 с ограничением до 255.


Морфологические операции выполняются с помощью структурирующего элемента, который последовательно перемещается по изображению. Базовыми морфологическими операциями являются эрозия и дилатация.

Для бинарного изображения дилатация белого объекта определяется так: если в окне присутствует хотя бы одна единица, то центральный пиксель принимает значение 1. В данной работе рассматривается расширение чёрного объекта на белом фоне, поэтому используется двойственная операция: изображение инвертируется, выполняется дилатация белых областей, затем выполняется обратная инверсия.

Для полутонового изображения дилатация определяется как замена центрального пикселя максимальным значением яркости в окне 3×3.

Эффект дилатации:
- увеличение размеров объекта;
- уменьшение внутренних отверстий и узких каналов;
- расширение ярких областей для полутонового изображения;
- слияние близко расположенных элементов.

## Практическая часть

**Принцип работы программы**
- Бинарные изображения
- Изображение загружается в градациях серого.
- Выполняется бинаризация.
- Выполняется дилатация чёрного объекта квадратом 3×3.
- Сохраняется результат в папку output/results_binary_dilated/.
- Строится XOR-разность между исходным и обработанным изображением.
- XOR-разность сохраняется в output/results_binary_diff/.
- Полутоновые изображения
- Изображение загружается в градациях серого.
- Выполняется дилатация как выбор максимума в окне 3×3.
- Результат сохраняется в output/results_semitone_dilated/.
- Вычисляется модуль разности между исходным и обработанным изображением.
- Дополнительно строится усиленная версия разности.
- Разности сохраняются в output/results_semitone_diff/.
- Формирование имён файлов

Имена выходных файлов формируются автоматически на основе имени исходного изображения.

Примеры:

test.png → test.dilated.png
test.png → test.xor_diff.png
cartoon.png → cartoon.abs_diff.png
cartoon.png → cartoon.abs_diff_contrast.png

## Описание обработки

### Часть 1. Бинарные изображения

#### Исходные данные

Бинарные изображения брались из папки:

`input/pictures_binary_src/`

#### Что формировалось

Для каждого бинарного изображения автоматически создавались:

- результат дилатации;
- XOR-разность между исходным и обработанным изображением.

#### Куда сохранялись результаты

Результаты дилатации сохранялись в папку:

`output/results_binary_dilated/`

Разностные изображения XOR сохранялись в папку:

`output/results_binary_diff/`

#### Особенности обработки

Для бинарных изображений дилатация применялась к чёрным объектам на белом фоне.  
Имена выходных файлов формировались автоматически на основе имени исходного изображения.

#### Вывод по части 1

На бинарных изображениях дилатация работает наглядно: тёмные объекты становятся толще, мелкие разрывы частично заполняются, отдельные элементы лучше соединяются между собой. Это хорошо видно на тексте, простых фигурах и изображениях с чёткими линиями.  
Одновременно с полезными объектами могут усиливаться и мелкие шумы, поэтому на некоторых изображениях результат выглядит грубее исходного.


### Часть 2. Полутоновые изображения

#### Исходные данные

Полутоновые изображения брались из папки:

`input/pictures_semitone_src/`

#### Что формировалось

Для каждого полутонового изображения автоматически создавались:

- результат дилатации;
- модуль разности между исходным и обработанным изображением.

#### Куда сохранялись результаты

Результаты дилатации сохранялись в папку:

`output/results_semitone_dilated/`

Разностные изображения сохранялись в папку:

`output/results_semitone_diff/`

#### Особенности обработки

Для полутоновых изображений дилатация выполнялась как выбор максимального значения яркости в окне `3×3`.  
Так как изменения после обработки не всегда хорошо заметны визуально, дополнительно строилось разностное изображение.

#### Вывод по части 2

На полутоновых изображениях дилатация в первую очередь усиливает светлые участки и меняет области перехода яркости. По самому результату это заметно не всегда сильно, особенно если изображение содержит плавные переходы и много деталей.  
Разностное изображение показывает эффект лучше: на нём хорошо видны границы объектов, контуры и участки с выраженной текстурой.


## Автоматизация сохранения результатов

Программа не требует ручного задания имён выходных файлов.  
Для каждого изображения имя результата формируется автоматически на основе имени исходного файла, а сам результат сохраняется в нужную папку в зависимости от типа изображения и вида обработки.

Это позволяет:

- не переименовывать файлы вручную;
- автоматически раскладывать результаты по нужным каталогам;
- удобно сравнивать исходные и обработанные изображения.


## Общий вывод по лабораторной работе

В лабораторной работе была реализована морфологическая операция дилатации с квадратным структурирующим элементом `3×3` в соответствии с вариантом 14. Программа автоматически обрабатывает бинарные и полутоновые изображения, сохраняет результаты по отдельным папкам и формирует разностные изображения.

Проверка на разных изображениях показала, что дилатация действительно увеличивает объекты, утолщает контуры и уменьшает небольшие разрывы. На бинарных изображениях это видно сразу и без дополнительных пояснений. На полутоновых изображениях эффект слабее заметен на глаз, поэтому разностное изображение здесь особенно полезно.
